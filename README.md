# H
very coolest game i've ever made.

![demo](demo.gif)
## How to run:
    git clone https://github.com/andynolife/h-moving.git
    cd h-moving
    # this nix-shell if you using nix
    # nix-shell
    gcc h.c -o h -lncurses
    ./h
